# hexgame

React/TypeScript Multiplayer Hexxagon-clone Game

### Local Development

```bash
# Installation
git clone git@github.com:veocode/hexgame.git
cd hexgame
./deploy.sh install

# Running
npm run dev

# Building
./deploy.sh build
```

### Production

```bash
# Installation
git clone git@github.com:veocode/hexgame.git
cd hexgame

# Up
./deploy.sh up

# Down
./deploy.sh down
```
